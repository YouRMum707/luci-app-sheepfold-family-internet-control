# Планирование аварийно-полезных сайтов

Название настройки в интерфейсе:

- EN: `Access to emergency-useful sites`
- RU: `Доступ к аварийно-полезным сайтам`

Список аварийно-полезных сайтов нужен для ограниченного режима: у устройства нет обычного доступа в интернет, но оно может открыть небольшой набор полезных сайтов.

Это режим аварийно-полезного доступа: достаточно для нужных сервисов, но недостаточно для обычного развлекательного интернета.

Цель — оставить необходимые сервисы и не открыть фильмы, игры, социальные ленты и развлекательные платформы.

## Правила продукта

- Аварийно-полезные сайты должны быть выключены по умолчанию, пока пользователь сам их не включит.
- В приложении нужен один редактируемый список доменов, а не встроенные пресеты. Пресеты и фильтрующие списки уже есть в AdGuard Home; Sheepfold не должен дублировать эту задачу.
- Родитель должен иметь возможность отредактировать весь список.
- Лучше добавлять узкие домены и поддомены, а не большие родительские домены.
- В интерфейсе нужно объяснить ограничения: HTTPS, CDN, мобильные приложения, DoH, общие домены и кэш браузера могут мешать точной фильтрации.
- В список по умолчанию нельзя добавлять домены фильмов, игр, соцсетей, коротких видео, стриминга и магазинов приложений.

## Редактируемый стартовый список

При первом запуске интерфейс может предложить этот список, но родитель должен просмотреть и подтвердить его перед включением ограниченного доступа:

```text
gosuslugi.ru
esia.gosuslugi.ru
mos.ru
school.mos.ru
dnevnik.ru
ya.ru
2gis.ru
```

## Полезные домены для ручного добавления

Это не пресеты, а примеры для редактируемого списка.

### Госуслуги и идентификация

```text
gosuslugi.ru
esia.gosuslugi.ru
lk.gosuslugi.ru
pos.gosuslugi.ru
mos.ru
my.mos.ru
school.mos.ru
```

### Школа и образование

Этот список должен редактироваться, потому что школьные платформы отличаются по регионам.

```text
dnevnik.ru
login.dnevnik.ru
school.mos.ru
edu.gosuslugi.ru
uchi.ru
resh.edu.ru
```

### Поиск, карты и базовые инструменты

Использовать аккуратно: даже обычный поисковик может привести к развлекательному контенту.

```text
ya.ru
google.com
translate.google.com
maps.google.com
2gis.ru
```

### Подсказка про Яндекс

Не добавлять широкие домены Яндекса по умолчанию.

Текст подсказки:

```text
Сервисы Яндекса сложно разрешить узко. Например, Яндекс.Картам могут понадобиться широкие домены Яндекса и общие статические домены. Если добавить yandex.ru, можно открыть не только карты, но и видео, музыку, игры, ленты, развлекательные страницы и другие сервисы Яндекса. Для простого поиска лучше использовать ya.ru, для карт — 2gis.ru. Более широкие домены Яндекса добавляйте только вручную и осознанно.
```

Яндекс.Карты полезны, но это не лучший домен для строгого стартового списка. В браузере им могут понадобиться широкие домены Яндекса и общие статические домены. Если добавить слишком широкий домен, можно открыть больше, чем только карты.

По умолчанию лучше оставить `2gis.ru`. Если родитель явно хочет Яндекс.Карты, можно попробовать добавить:

```text
maps.yandex.ru
yandex.ru
yandex.com
yastatic.net
api-maps.yandex.ru
static-maps.yandex.ru
```

Не стоит добавлять широкие домены Яндекса по умолчанию:

```text
yandex.ru
video.yandex.ru
kinopoisk.ru
music.yandex.ru
games.yandex.ru
dzen.ru
```

Если нужен поиск Яндекса, лучше сначала предложить `ya.ru`, а более широкие домены оставить на явное решение родителя.

### Банки

Доступ к банкам может быть полезен для родителей или бытовых задач, но его не стоит включать в строгий детский список по умолчанию. Банковские мобильные приложения могут использовать дополнительные API и CDN-домены, которые отличаются от браузерных доменов.

Домены для проверки в браузере:

```text
sberbank.ru
www.sberbank.ru
sberbank.com
www.sberbank.com
online.sberbank.ru
tbank.ru
www.tbank.ru
business.tbank.ru
cdn-tinkoff.ru
alfabank.ru
click.alfabank.ru
web.alfabank.ru
```

### Связь и безопасность

Опционально. Это может быть полезно, но также может стать способом обхода ограничений.

```text
mail.google.com
mail.yandex.ru
web.whatsapp.com
telegram.org
```

Этот блок не нужно включать по умолчанию.
