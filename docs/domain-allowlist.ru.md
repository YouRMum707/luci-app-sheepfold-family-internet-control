# Планирование белого списка доменов

Белый список доменов нужен для ограниченного режима: у устройства нет обычного доступа в интернет, но оно может открыть небольшой набор полезных сайтов.

Цель — оставить необходимые сервисы и не открыть фильмы, игры, социальные ленты и развлекательные платформы.

## Правила продукта

- Белый список доменов должен быть выключен по умолчанию, пока пользователь сам его не включит.
- В приложении лучше дать рекомендуемые пресеты, а не жёсткий список навсегда.
- Родитель должен иметь возможность отредактировать любой пресет.
- Лучше добавлять узкие домены и поддомены, а не большие родительские домены.
- В интерфейсе нужно объяснить ограничения: HTTPS, CDN, мобильные приложения, DoH, общие домены и кэш браузера могут мешать точной фильтрации.
- В список по умолчанию нельзя добавлять домены фильмов, игр, соцсетей, коротких видео, стриминга и магазинов приложений.

## Рекомендуемые пресеты

### Госуслуги и идентификация

Для документов, государственных сервисов и авторизации.

```text
gosuslugi.ru
esia.gosuslugi.ru
lk.gosuslugi.ru
pos.gosuslugi.ru
mos.ru
my.mos.ru
school.mos.ru
```

### Школа и образование

Этот список должен редактироваться, потому что школьные платформы отличаются по регионам.

```text
dnevnik.ru
login.dnevnik.ru
school.mos.ru
edu.gosuslugi.ru
uchi.ru
resh.edu.ru
```

### Поиск, карты и базовые инструменты

Использовать аккуратно: даже обычный поисковик может привести к развлекательному контенту.

```text
ya.ru
google.com
translate.google.com
maps.google.com
2gis.ru
```

Не стоит добавлять широкие домены Яндекса по умолчанию:

```text
yandex.ru
video.yandex.ru
kinopoisk.ru
music.yandex.ru
games.yandex.ru
dzen.ru
```

Если нужен поиск Яндекса, лучше сначала предложить `ya.ru`, а более широкие домены оставить на явное решение родителя.

### Связь и безопасность

Опционально. Это может быть полезно, но также может стать способом обхода ограничений.

```text
mail.google.com
mail.yandex.ru
web.whatsapp.com
telegram.org
```

Этот пресет не нужно включать по умолчанию.

## Рекомендация по умолчанию

Не включать домены автоматически.

В интерфейсе предложить пресеты:

- Госуслуги;
- Школа;
- Поиск и карты;
- Связь;
- Свои домены.

Самая аккуратная стартовая рекомендация:

```text
gosuslugi.ru
esia.gosuslugi.ru
mos.ru
school.mos.ru
dnevnik.ru
ya.ru
2gis.ru
```

Родитель должен просмотреть и подтвердить список перед включением.
