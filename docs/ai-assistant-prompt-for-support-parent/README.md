# Здесь есть промпты которые нужны для настройки ассистента в мобильном приложении, их основная цель - помочь родителям с налаживанием взаимодействий с детьми.

## Структура
- v1/ — оригинальный промпт из проекта (далал Codex)
- v2/ — расширенная версия с поддержкой родителей (далал Grok)

Используй самую большую версию для более empathetic и разностороннего помощника.
